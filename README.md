
# 《御灵》 - 2D 修真养成游戏

## 个人博客-御灵项目介绍--跳转链接
https://copejay.github.io/project/game1.html

## 项目简介
独立开发的 2D 修真题材养成游戏，采用 DDD 分层架构设计。
抽奖，战斗获取资源，背包管理资源，角色培养消耗资源，实现资源循环利用

## 技术栈
- Cocos Creator 3.8
- TypeScript
- DDD / 模块化
- 数据驱动配置

## 核心系统（实现资源循环）
- 战斗系统
- 背包系统
- 角色培养系统
- 大地图系统

## 项目规模
- 核心逻辑代码约 1.2 万行
