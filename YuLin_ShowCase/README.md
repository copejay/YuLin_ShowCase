
# 《御灵》 - 2D 修真养成游戏

## 项目简介
独立开发的 2D 修真题材养成游戏，采用 DDD 分层架构设计。

## 技术栈
- Cocos Creator 3.8
- TypeScript
- DDD / 模块化
- 本地存档系统
- 数据驱动配置

## 架构设计
详见 docs/architecture.png

## 核心系统
- 战斗系统
- 背包系统
- 角色培养系统
- 大地图系统

## 项目规模
- 核心逻辑代码约 1.2 万行
- 配置数据约 2k 行
